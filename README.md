# `react-default-jsx-app`

## Setup

### First time

```bash
$> npm run setup
```

It mainly installs global dependencies

### Dependencies

```bash
$> npm install
```

## R&D

### Test

`FIXME` not working

```bash
$> npm test
```

### Build

#### `build/`

```bash
$> npm run build
```

#### `out/`

```bash
$> npm run out
```

### Run

```bash
$> npm start
```

This will host the app in http://localhost:3000

## `electron`

### Run

```bash
$> npm run electron
```

### `forge`

#### Package

```bash
$> npm run electron:forge:package
```

#### Make

```bash
$> npm run electron:forge:make
```
